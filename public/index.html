🎓 DBMS Concepts Demonstrated

NoSQL

Embedded Documents

skills[] lives inside each Job document — no junction table needed unlike SQL's job_skills table.

Pipeline

$unwind Aggregation

Deconstructs the skills array: one job with 5 skills becomes 5 documents, then grouped and counted.

Index

Multikey Index

Index on skills[] turns COLLSCAN → IXSCAN. See the Explore DB tab for a live demo.

Redundancy

Controlled Redundancy

Application stores job_title and applicant_name to eliminate $lookup joins — NoSQL's read optimization.

📊 Skill Demand Analysis

$unwind → $group → $sort

Deconstructs each job's skills[] array, groups by skill name, and counts market demand across 5,000+ listings.

View Pipeline Code 
[
  { $unwind: "$skills" },
  { $group: { _id: "$skills", count: { $sum: 1 } } },
  { $sort: { count: -1 } },
  { $limit: 15 }
]

📈 Application Funnel

$group → $sort

Groups all applications by their status to visualise the hiring funnel — from Applied to Offered or Rejected.

View Pipeline Code 
[
  { $group: { _id: "$status", count: { $sum: 1 } } },
  { $sort: { count: -1 } }
]

🏢 Jobs Per Company

$group → $sort

Groups jobs by company name, counting listings and calculating average salary. Sorted by total job count descending.

View Pipeline Code 
[
  { $group: { _id: "$company", totalJobs: { $sum: 1 }, avgSalary: { $avg: "$salary" } } },
  { $sort: { totalJobs: -1 } },
  { $limit: 12 }
]

💰 Salary by Experience

$match → $group → $sort

Aggregates salary statistics grouped by experience level — shows the average, minimum, and maximum salary at each career stage.

📍 Jobs by City

$group → $sort → $limit

Groups job postings by city and ranks the top 10 hiring hubs in India, with average salary overlay.
